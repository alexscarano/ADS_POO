# ADS_POO

Repositório dedicado à disciplina de **Programação Orientada a Objetos (POO)**.

## Tecnologias Utilizadas

- **Java**
- **HTML**

## Sobre

Este projeto contém exemplos, exercícios e projetos aplicados na disciplina de POO, abrangendo conceitos fundamentais como encapsulamento, herança, polimorfismo e interfaces.

## Como executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/alexscarano/ADS_POO.git
   ```
2. Importe os arquivos Java em sua IDE favorita (Eclipse, IntelliJ, NetBeans).
3. Execute os exemplos conforme o README de cada pasta/modulo.
